HOW to Run :

1. After Cloning this Repository, run 
`````````
npm install 
`````````
2. based on folder feature, you can run anything based on tag with syntac 
`````````
npm run test -- --tag [tagname]
`````````
or for all test
````````
npm run test
````````
3. for generating report run script 
```````
npm run report-chrome
```````
